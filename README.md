#Project Clinic POO

Java backend application for managing a clinic: Patients, doctors, appointments and treatments

1.Functionalities:
-CRUD for patients, doctors and appointments
-Login for clients
-Integration Hibernate with MySQL

2.Technologies:
-Java
-Hibernate
-MySQL
-Maven

3.How to run the project:
-Clone the repo 
-configure MySQL database
-run the application (preferable in IntelliJ)
